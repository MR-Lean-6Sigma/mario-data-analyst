# Six Sigma Integration - DMAIC + ROI Analysis

## Overview

Complete Six Sigma improvement project demonstrating the DMAIC methodology with financial ROI analysis. This is the capstone project that integrates everything you've learned and showcases your manufacturing engineering expertise.

## What is DMAIC?

DMAIC is the Six Sigma improvement process that Mario learned as a Green Belt and is now implementing:

1. **Define**: Identify the problem and business impact
2. **Measure**: Collect data and establish baseline metrics
3. **Analyze**: Find root causes using statistical analysis
4. **Improve**: Design and implement solutions
5. **Control**: Monitor results and sustain improvements

## Project Structure

This project demonstrates a complete 6 Sigma improvement initiative:

### Phase 1: DEFINE
- Problem statement (defects in production line)
- Business impact analysis
- Project scope and objectives
- Financial targets

### Phase 2: MEASURE
- Collect baseline data
- Calculate baseline metrics (DPMO, Sigma Level)
- Establish measurement system

### Phase 3: ANALYZE
- Statistical analysis of defects
- Root cause analysis using Pareto charts
- Process capability analysis (Cp, Cpk)
- Hypothesis testing

### Phase 4: IMPROVE
- Design solutions
- Calculate expected improvements
- Estimate implementation costs

### Phase 5: CONTROL
- Monitor new process performance
- Calculate actual improvements achieved
- Calculate net ROI of project
- Create control plan for sustainability
