### 4. Six Sigma Green Belt DMAIC - Complete Project
- **Folder:** `Green-Belt-DMAIC/`
- **Files:** 21 comprehensive project files
- **Duration:** July 2020 - Project Completion
- **Scope:** Defect reduction in packaging/guillotine cutting process
- **Methodology:** Define-Measure-Analyze-Improve-Control
- **Status:** ✅ CERTIFIED PROJECT

**The Complete DMAIC Journey:**
- **DEFINIR (Define):** Problem identification and scoping
- **MEDIR (Measure):** Data collection and baseline establishment
- **ANALIZAR (Analyze):** Root cause identification
- **MEJORAR (Improve):** Solution implementation and 5S
- **CONTROLAR (Control):** Sustainability and monitoring

**Key Deliverables:**
- Project Charter with business case
- SIPOC analysis (Supplier-Input-Process-Output-Customer)
- Process maps and flow analysis
- Data collection methodology
- Statistical baseline analysis
- Ishikawa (Fishbone) diagrams
- FMEA (Failure Mode & Effects Analysis)
- Root cause analysis (5 Whys)
- 5S implementation in production area
- Control charts showing improvement
- Procedures and sustainability plans

**Documentation:**
Each DMAIC phase has:
- Working Excel files showing actual analysis
- Gate Review approvals
- README explaining the phase
- Deliverables and outcomes
