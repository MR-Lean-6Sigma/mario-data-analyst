# FASE 4: MEJORAR (Improve)

## Objetivo

Diseñar e implementar soluciones específicas a las causas raíz identificadas. Verificar que las mejoras funcionan y generan el impacto esperado.

### 03-5S_IMPLEMENTACION.pptx
**Implementación de 5S - Embalaje y Guillotina**

Presentación visual mostrando:

**5S Program (Lean - Five Sorting Steps):**

1. **Sort (Clasificar/Seiri)**
   - Identificar lo necesario vs. innecesario
   - Remover herramientas/materiales que no se usan
   - Limpiar el espacio

2. **Set in Order (Organizar/Seiton)**
   - Lugar para cada cosa
   - Fácil acceso a herramientas frecuentes
   - Señalización clara
   - Organización lógica

3. **Shine (Limpiar/Seiso)**
   - Limpiar máquinas y área
   - Inspeccionar durante limpieza
   - Identificar desgaste

4. **Standardize (Estandarizar/Seiketsu)**
   - Documentar procedimientos
   - Crear checklists visuales
   - Fotos de estado ideal
   - Standards de limpieza

5. **Sustain (Mantener/Shitsuke)**
   - Entrenar equipo
   - Auditorías periódicas
   - Disciplina para mantener
   - Mejora continua

**Propósito:** Crear ambiente ordenado, seguro, eficiente

- Cambio de cuchillas
