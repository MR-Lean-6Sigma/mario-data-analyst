## 📋 Project Overview

Complete **DMAIC (Define-Measure-Analyze-Improve-Control)** project demonstrating the application of Six Sigma methodology to a real manufacturing process improvement initiative.

This project addresses defect reduction in a production line packaging and guillotine cutting process, applying rigorous statistical and analytical tools to identify root causes and implement sustainable improvements.

---

## 🎯 Project Phases

### [DEFINE] - Project Charter & Problem Definition

Establishes the project foundation by defining the business problem, scope, and objectives.

**Deliverables:**
- **Project Charter** - Business case, objectives, scope, timeline, resources
- **SIPOC Diagram** - Supplier-Input-Process-Output-Customer mapping
- **Process Map** - Current state process visualization
- **Gate Review** - Project approval and sign-off

**Key Outcomes:**
- Clear problem statement
- Defined project scope
- Established baseline metrics
- Project team alignment

---

### [MEASURE] - Baseline Data Collection & Analysis

Collects empirical data on current process performance and establishes statistical baseline.

**Deliverables:**
- **Data Collection Plan** - Measurement approach and data requirements
- **Sequential Control Chart** - Time-series trend visualization
- **Gate Review** - Measurement system validation

**Key Metrics:**
- Baseline defect rate
- Process variation
- Process capability (Cp, Cpk)
- Data integrity verification

**Statistical Tools Used:**
- Control Charts (I-MR)
- Descriptive Statistics
- Normality Testing

---

### [ANALYZE] - Root Cause Analysis

Deep investigation to identify the vital few causes driving the majority of defects.

**Deliverables:**
- **Ishikawa Diagram 1** - Comprehensive cause-effect diagram (Fishbone)
- **Ishikawa Diagram 2** - Prioritization matrix (impact vs. effort)
- **Ishikawa Diagram 3** - 5 Whys analysis and action plan
- **FMEA (Failure Mode & Effects Analysis)** - Risk assessment
- **Gate Review** - Root cause validation

**Analysis Tools:**
- Fishbone Diagram (Ishikawa)
- 5 Whys Analysis
- Pareto Analysis
- FMEA (Failure Mode & Effects Analysis)
- Prioritization Matrix

**Root Causes Identified:**
1. Process variation in cutting parameters
2. Inconsistent operator procedures
3. Inadequate visual controls
4. Environmental factors (humidity, temperature)
5. Material inconsistency

---

### [IMPROVE] - Solution Design & Implementation

Develops and implements solutions to address root causes, focusing on sustainability and sustainability.

**Deliverables:**
- **Cause-Effect Matrix** - Solution selection based on cause-effect relationships
- **FMEA Final** - Risk assessment of new process design
- **5S Implementation** - Embalaje (Packaging) and Guillotina (Guillotine) area
  - Sort (Clasificar) - Remove unnecessary items
  - Set in Order (Organizar) - Organize workspace
  - Shine (Limpiar) - Clean and inspect
  - Standardize (Estandarizar) - Create standards
  - Sustain (Mantener) - Maintain discipline
- **Gate Review** - Solution approval
- **June Control Chart** - Initial improvement verification

**Improvements Implemented:**
- Standardized work procedures
- Visual control systems
- Operator training program
- Tool and material organization
- Environment monitoring

**Results (June):**
- Defect reduction trend evident
- Process variation decreased
- Control chart shows improvement trajectory

---

### [CONTROL] - Sustain Improvements & Monitor Long-term

Ensures improvements are sustained and embedded in standard operations.

**Deliverables:**
- **Control Plan** - Monitoring strategy, control methods, reaction plan
- **Standard Operating Procedures (SOP)** - Documented standard work
- **Gate Review** - Project closure and lessons learned
- **July Control Chart** - Long-term sustainability verification

**Sustainability Measures:**
- Preventive maintenance schedule
- Operator certification program
- Periodic audits and reviews
- Root cause prevention system
- Visual management boards

**Final Results (July):**
- Sustained defect reduction
- Process control established
- Continuous monitoring active
- Team capability developed
