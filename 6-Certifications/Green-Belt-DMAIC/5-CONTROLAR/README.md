# FASE 5: CONTROLAR (Control)

## Objetivo

Confirmar que las mejoras se sostienen a largo plazo. Implementar sistemas de control para mantener los ganoos y prevenir regresión al estado anterior.

---

## 📄 Archivos en Esta Fase

### 01-PLANO_CONTROL.xlsx
**Control Plan - Plan de Monitoreo a Largo Plazo**

Documento que define:

**Para cada Critical-to-Quality (CTQ):**
- **Qué medir:** Variable a monitorear
- **Cómo medir:** Método de medición
- **Frecuencia:** Cada hora, diaria, semanal
- **Control Limit:** Tolerancia aceptable
- **Quién:** Responsable de medición
- **Acción:** Qué hacer si sale de control

**Mecanismos de Control:**
- Control charts continuos
- Checklists visuales
- Auditorías periódicas
- Reuniones de revisión

**Propósito:** Mantener vigilancia permanente del proceso

---

### 02-PROCEDIMIENTO_OPERACIONAL.xlsx
**Standard Operating Procedure (SOP) - Trabajo Estandarizado**

Documentación completa mostrando:

**Para cada paso del proceso:**
- Procedimiento paso a paso
- Parámetros de máquina (velocidad, presión, temperatura)
- Especificaciones de entrada
- Criterios de aceptación
- Qué hacer si hay anormalidad
- Fotos o diagramas de referencia

**Secciones típicas:**
- Setup inicial
- Ciclo operacional
- Inspección en proceso
- Cierre/cambio de producto
- Troubleshooting

**Propósito:** Garantizar consistencia operacional día a día

---

### 03-GATE_REVIEW.xlsx
**Cierre de Proyecto - Gate Review Final**

Aprobación formal del cierre:

- ✓ Mejoras sustentadas (Junio + Julio)
- ✓ Métricas de éxito alcanzadas
- ✓ SOP documentado y entrenado
- ✓ Control plan implementado
- ✓ Beneficios financieros confirmados
- ✓ Equipo empoderado
- ✓ Lecciones documentadas
- ✓ Proyectos futuros identificados

**Resultado:** Proyecto oficialmente CERRADO Y CERTIFICADO

---

### 04-GRAFICO_CONTROL_JULIO.xlsx
**Control Chart - Julio (Mes 2 Post-Mejora)**

Gráfico secuencial mostrando:
- Performance en mes 2 después de mejoras
- ¿Se sostiene la mejora de Junio?
- ¿Mejoró más o se estabilizó?
- ¿La variación permanece baja?

**Comparación Junio vs. Julio:**
- Si continúa bajando → Mejora sostenida
- Si se estabiliza → Proceso bajo control
- Si sube → Problema de sostenibilidad

**Criterio de Éxito:**
- Defect rate significativamente menor al baseline
- Proceso estable (sin cambios erráticos)
- Dentro de especificaciones

**Propósito:** Validar que mejoras son permanentes
